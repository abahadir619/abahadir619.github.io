    Now we will go further with TROOFS and creat them with comparison statements! Comparison statements require two or more variables and compare them using different operators such as:
    BOTH SAEM X AN Y = true if x and y are equal
    DIFFRINT X AN Y = true if x and y not equal
    BOTH SAEM X AN BIGGR OF X AN Y = true if x >= y
    BOTH SAEM X AN SMALLR OF X AN Y = true if x <= y
    DIFFRINT X AN SMALLR OF X AN Y = true if x > y
    DIFFRINT X AN BIGGR OF X AN Y = true if x < y
